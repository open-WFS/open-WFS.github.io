Open Wave Field Synthesis (OpenWFS)

Wave field synthesis is a spatial audio rendering technique that uses high-density loudspeaker arrays to reproduce the pressure fronts of spatially-distributed sound sources, creating realistic three-dimensional sound scenes.

Open Wave Field Synthesis (OpenWFS) is a new, modular system for wave field synthesis that is designed for experimentation and prototyping new approaches to working with spatial sound.

Features

  • Lifelife 3D spatial audio rendering: Capable of rendering eerily realistic sound fields using wavefield synthesis and beamforming algorithms
  • Modular and flexible: Can be used with existing workflows and packages such as Spat for Max
  • Designed for experimentation: Completely open APIs and rendering stack, allowing for the development and experimentation using custom algorithms
  • Rapid, friction-free setup: Audio distribution via AVB means that each module of 32 channels requires only power and a single CAT6 cable
